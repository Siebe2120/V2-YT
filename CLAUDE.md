# V2-YT

This repo is deployed to Vercel (https://v2-yt-pi.vercel.app/), auto-deploying from the `main` branch on every push.

## Workflow

- After making a requested code change, commit it and push to `origin/main` without asking for confirmation first — the user has authorized this so changes go live on Vercel automatically. Still use clear commit messages and avoid force-pushing or rewriting history.
